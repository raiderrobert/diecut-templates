# diecut-templates

Starter templates for [diecut](https://github.com/raiderrobert/diecut).

## Templates

| Template | Description | Command |
|----------|-------------|---------|
| [python-pkg](python-pkg/) | Python package with src layout and hatchling | `diecut new gh:raiderrobert/diecut-templates/python-pkg` |
| [rust-cli](rust-cli/) | Rust CLI application | `diecut new gh:raiderrobert/diecut-templates/rust-cli` |
| [static-site](static-site/) | Static HTML site | `diecut new gh:raiderrobert/diecut-templates/static-site` |

## Usage

Install [diecut](https://github.com/raiderrobert/diecut#install), then:

```bash
diecut new gh:raiderrobert/diecut-templates/rust-cli --output my-project
```

Use `--defaults` to skip prompts and accept default values.
